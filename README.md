# 🧾 Shabbat Planner App

A React application for managing preparation workflows using shopping lists, cooking plans, and task tracking — all in one place.

Built with **React + Vite**, the app combines multiple data sources and user settings to generate dynamic preparation lists with persistent progress.

---

## ✨ Features

- 🛒 Dynamic shopping lists (base, guests, travel)
- 🍳 Cooking planner by meals
- ✅ Home task management
- ⚙️ Configurable settings
- 💾 Automatic LocalStorage persistence
- 📊 Progress tracking for shopping, tasks, and cooking

---

## 🛠️ Tech Stack

- React
- Vite
- Context API
- JavaScript (ES6+)
- CSS
- LocalStorage
- UUID
